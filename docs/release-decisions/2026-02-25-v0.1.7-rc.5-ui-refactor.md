# Decisao de Release - v0.1.7-rc.5 (UI refactor)

## Data

- 2026-02-25

## Objetivo

Publicar uma release candidata com a refatoracao visual/estrutural da UI do Dexter para shell em 3 colunas, mantendo contratos existentes de operacao local (runtime/modelos/update/governanca) e automacoes E2E.

## Escopo incluido

- Shell React em 3 colunas (`ActivityBar`, `Workspace`, `SidePanel`).
- Contextos de sidepanel por view (`chat`, `modules`, `settings`, `governance`).
- `ModuleManager` com separacao de `Core System`, instalados e descoberta.
- `SettingsPanel` em accordion.
- `GovernancePanel` consolidado (permissoes, update, auditoria/exportacao).
- Migracao incremental de listeners para intents UI (`ui-intents`) com controle legado em `legacy-main.ts`.
- Atualizacao de testes E2E/visuais para novo fluxo de navegacao.
- Atualizacao da documentacao arquitetural/operacional para refletir o layout e bootstrap atuais.

## Evidencias locais

- `npm run typecheck`: ok
- `npm run test`: ok (239/239)
- `npm run build:renderer`: ok
- `npm run test:e2e`: ok (15/15)

## Riscos conhecidos

- A migracao para React ainda convive com camada legada (`legacy-main.ts`), exigindo disciplina para manter IDs/contratos estaveis enquanto a extracao avanca.
